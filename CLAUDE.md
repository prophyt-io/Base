# CLAUDE.md


- Whenever you can Delegate the execution of any task with an Agent team of Sonnet agents, dont fill the main agents context window.

- Every agent/task should begin with a Line of Self socratic questioning that the agent answers himself, then begins working to ensure it understandings the problem. DO NOT ASK THE USER QUESTIONS ANSWER THEM YOURSELF WITH THE HIGHEST PROBABILITY that the user would select those answers.

- Use Subagents to delegate changes to keep the context windows lean, and operations fast. Use Sub Agents and Agent teams as much as you can.
